# **App Name**: HealthWise Scan

## Core Features:

- User Health Profile: Input and store user's health details including conditions and allergies. Supports editing and saving profiles.
- AI Barcode Scanner: Scan product barcodes using the device camera or upload images to fetch product details like ingredients and nutritional info.
- AI Analysis & Alerts: Analyze product ingredients against the user's health profile using a tool to determine safety. Display safety status with color-coded alerts and AI-generated explanations.
- Smart Recommendations: Suggest 2-3 alternative products based on compatibility with the user's health profile.
- Voice Assistant: Enable voice commands to scan products, check safety, and get alternative suggestions, with AI-generated voice responses.
- Chatbot Health Assistant: Provide a chatbot interface where users can ask health-related questions and receive AI-driven answers.
- Dashboard Analytics: Display charts and analytics showing safe vs. unsafe scans and daily nutrient intake.

## Style Guidelines:

- Primary color: Soft green (#A7D9A9) to convey health and wellness.
- Background color: Light white (#F7F9F7), a very desaturated tone of soft green.
- Accent color: Light yellowish-green (#C6E273), an analogous color to the primary.
- Body and headline font: 'PT Sans' for a modern, accessible feel.
- Use health-related icons for health profiles, scanning, chatbot, and safety alerts.
- Employ rounded cards and subtle shadows to enhance depth and modern aesthetics.
- Incorporate smooth animations for transitions and loading states to improve user experience.
